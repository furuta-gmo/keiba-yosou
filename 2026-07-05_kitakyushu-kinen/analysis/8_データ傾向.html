データ傾向（北九州記念の過去傾向）

人気・枠・脚質・斤量・年齢・ローテを実数で精査／ 確信度: 中〜高・重み6%

北九州記念は「激荒れ」重賞——過去10年で勝ち馬の半数が6番人気以下、3連単は全て万馬券（10万超4回）。今年の該当馬に当てはめると1〜2番人気ともデータ上の赤信号、トップハンデ級と8歳は割引、妙味は5〜9番人気の前めローテ良化馬にある、と実数が示す。

1. 人気別（激荒れ）

  • 1番人気 1-3-1-5（勝率10%/複勝50%）＝連軸には使えるが勝ち切れない＝単勝1着付け危険 → ⑫フリッカージャブに該当
  • 2番人気 0-0-1-9（過去10年"最不振"の人気帯） → ⑦デアヴェローチェに警告データ
  • 6〜9番人気 4-1-3-32（勝4）／10番人気以下も勝利例（16番人気勝ちあり）＝人気薄が突っ込む

2. 枠・脚質

  • 枠：5〜8枠（中外）から勝ち馬集中、4枠は勝ち星ゼロ、内枠不振（20年では2枠・8枠が上位、7枠不振）。
  • 脚質：差し5勝が最多＋逃げ2・先行2。過去5年で馬券圏15頭中10頭が4角6番手以内＝前めが近道だが、勝ち切るのは差し。追込一辺倒は割引。→ ①展開の「ハナ争い激化→差し台頭」と符合。

3. 斤量・年齢（今年の該当馬に直結）

  • 57kgちょうどは優秀（連対66.7%）だが、57.5以上＝トップハンデ級は勝率1.6%＝勝ち切れない → ⑫フリッカー57.5・⑪ヤマニン57.5・⑨ヨシノ58に割引
  • 軽ハンデ51〜53からも複数勝利＝⑧アメリカン51・②ジェニファー50・⑦デア/⑬イツモ/③プロト53に恩恵
  • 斤量増ローテ◎（前走1着で斤量増＝複勝58.8%）／前走距離短縮組＝0-0-0-17で全滅 → ①ランフォー（マイル→1200短縮）は消し材料
  • 3歳（複勝26%）×牝馬（複勝23%）は好走圏 → ⑦デア（3歳牝）は本質データ好走圏（但し2番人気の壁）
  • 8歳＝0-0-0-6で完全消し → ⑨ヨシノイースター（8歳）は当レース好走歴があっても年齢は逆風

4. 前走ローテ

  • 前走1着馬が断然（5-7-3-22）「夏は勢い重視」 → ⑫フリッカー(鞍馬S1着)・⑦デア(葵S1着)
  • CBC賞・中京開催組は割引（複勝10.7%）。2026年のCBC賞は中京開催＝CBC組は評価下げ、別路線を狙うのがデータに沿う

5. 道悪の年（サンプル少）

直近10年で稍重開催は2020・2021の2年のみ。この2年は逃げ馬が馬券圏に残った＝道悪＝前残り強化の傾向。ただしサンプル僅少で参考値。重・不良の確定統計は取れず（→④コース馬場の二面性で補完）。

6. 結論

データは「上位人気3頭はいずれも頭固定に不安（1番人気勝ち切れず・2番人気最不振・トップハンデ級と8歳割引）、勝ち馬は5〜9番人気の前めローテ良化馬」と示す。今年の妙味の型は「⑫⑦を軸で信頼しきらず、軽ハンデ・道悪実績のある5〜9番人気（⑤アブキール・⑧アメリカン、道悪なら⑥⑬）を絡めた相手厚めの馬券」。データ傾向は重み6%と低いが、今回は複数の実数（2番人気最不振・トップハンデ1.6%・8歳消し・距離短縮全滅）が該当馬に直接刺さり、判断を後押しする。

出典: SPREAD（過去10年）・ウマニティ（過去20年）・Winsight・m-jockey・SPAIA（ローテ/斤量）ほか。過去10年と20年で母数が異なり脚質・枠の見え方に差がある点、道悪限定の確定成績表は未取得の点に留意。
