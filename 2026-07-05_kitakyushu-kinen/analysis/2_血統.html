血統（父・母父・道悪適性）の分析

全13頭の父/母父を実確認＋小倉芝1200m重不良の血統データで道悪適性を評価／ 確信度: 中〜高（血統は実確認、個体の道悪実績は一部未確認）

道悪(重〜不良)濃厚な今回、血統分析の主眼は「小倉芝1200mの道悪で走る血統か」。実データ(小倉芝1200m・重/不良の馬券内回数)ではビッグアーサー24回・ダイワメジャー21・ロードカナロア18・ファインニードル10が上位＝パワー×スピードの短距離血統が効く。この「道悪巧者血統」が人気薄サイドに集中し、逆に1番人気⑫フリッカージャブは芝道悪未経験＋スピード馬場向きの母系——ここに血統由来の妙味構造がある。

2. 小倉芝1200m「重・不良」の血統データ（実測）

系統別ではミスプロ系が最強、次いでダンジグ系。馬券内回数の上位種牡馬——純粋なパワー型の短距離スピード血統が効き、良馬場の高速決着(瞬発力型)とは要求が変わる。ディープ系の軽い芝向きはむしろ割引。

  • ビッグアーサー … 馬券内24回（最多） → 該当: ⑬イツモニコニコ(父)
  • ダイワメジャー … 21回 → 該当馬なし
  • ロードカナロア … 18回 → 該当: ①ランフォーヴァウ(父)※但し距離短縮で割引
  • ファインニードル … 10回 → 該当: ⑤アブキールベイ(父)
  • ミスプロ系全般 → ⑥オタルエバー(父リオンディーズ=キンカメ系)、⑪ヤマニン(母父ミスプロ)

※小倉の含水率基準(芝): 良10%以下/稍重8-12%/重10-14%/不良12%以上。含水が上がるほど時計を要し、パワーとスタミナが問われる。出典: JRA含水率解説・「ゼロから始める競馬入門(小倉重不良血統)」・うまめし.com ほか。

4. 結論

道悪(重〜不良)になれば、血統・実データは「1番人気⑫フリッカーの道悪適性は未証明で割引、妙味は道悪血統・実績の人気薄サイドに集中」と明確に示す。道悪実績で最も堅いのは⑥オタルエバー(重2勝＝メンバー随一)と⑬イツモニコニコ(ビッグアーサー×稍重1200実績)、次いで⑤アブキールベイ(ファインニードル×重5着＋好騎手)。人気サイドでは⑦デアヴェローチェ(稍重1着＋川田)が道悪の裏付けを持つ数少ない存在＝相手の中心。逆に④アンクルは道悪不良8着凡走で過大評価しない。血統は今回、勝負の中核情報——馬場が渋るほど、この序列が効いてくる。

※父・母父はnetkeiba/JBIS/競馬ラボで実確認。種牡馬の道悪傾向は各産駒データ・小倉重不良集計に基づく。個体の道悪実績が「未確認」の馬(⑩⑪等)は血統推定であり、当日の馬場発表＋各馬の道悪実績最終確認で補完すること。⑧アメリカンビキニの芝稍重は「先行2着」と「8着」でソース間に差があり要確認。極端な不良は全頭サンプル薄で「？」が残る。
