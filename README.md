# 競馬予想 (PWA)

5視点分析×期待値（回収率重視）で組み立てる予想サイト。スマホでホーム画面に追加すると全画面アプリ＆オフライン閲覧可。

- 公開URL: https://furuta-gmo.github.io/keiba-yosou/
- トップ: `index.html`
- 宝塚記念2026: `2026-06-14_takarazuka/prediction.html`
- オッズ計算機: `2026-06-14_takarazuka/odds.html`

## スマホから予想を更新する3つの方法

### 方法1：Claudeと連携してフル更新（推奨）
最新情報（オッズ・馬場急変・馬体重など）を入れて、買い目まで作り直したいとき。

1. スマホのブラウザで **claude.ai/code** を開く（または Claude アプリ）
2. このリポジトリ **furuta-gmo/keiba-yosou** に接続して開く
3. 下の「更新依頼テンプレ」を貼って送る
4. Claude が [CLAUDE.md](CLAUDE.md) の手順に従い、ファイル編集→`git push` まで実行
5. 1分ほどで GitHub Pages が再ビルド → アプリをオンラインで開き直すと最新化

**更新依頼テンプレ：**
```
【更新依頼】レース: 宝塚記念2026
種類: [オッズ / 馬体重 / 馬場 / 結果 / その他]
内容: 例) 直前にゲリラ豪雨で良→重に変化。最新単勝 5=1.5, 16=7.8 ...
希望: 予想を更新してpushして
```

### 方法2：オッズ計算機で即チェック（オフラインOK・Claude不要）
締切直前にオッズだけ見て妙味を確認したいとき。
`odds.html` で単勝を入力 → EV・妙味・相手比重が即再計算。入力は端末に保存。

### 方法3：PCのClaude Codeで頼む
PC作業中なら従来どおり「最新オッズで予想を更新して」で同じことができる。

## 重要メモ（更新時）
- 更新したら **`sw.js` の CACHE 版を上げる**（スマホ反映のため）。詳細は [CLAUDE.md](CLAUDE.md)。
- 買い目は **100円単位**。方針は回収率重視・期待値型。
